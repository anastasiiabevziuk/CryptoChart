## Build and Run the project


### `npm init`
### `npm run build`
### `npm run start`

now you can run [http://localhost:3000](http://localhost:3000) to view it in the browser.


## Open performance profile report 

### run Google chrome
### press F12 button
### go to performance tap
### open context menu
### click "load profile..."
### choose "PerformanceProfile.json"
